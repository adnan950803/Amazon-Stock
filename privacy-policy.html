9. GitHub Deployment Instructions

You can host this website using GitHub Pages by following these steps:

  • Create a GitHub account at github.com
  • Create a new repository named amazon-stock
  • Upload your HTML file (e.g., index.html)
  • Go to Settings → Pages
  • Select branch: main and folder: /root
  • Click Save and wait for the live link
